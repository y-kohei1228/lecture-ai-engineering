##はじめに
linuxコマンド集を共有したいと思います。
LinuxというOSがあり、さらにLinuxのディストリビューションが各種ありまして、それぞれ、コマンドが違います。
最初のLinuxコマンド集は一般企業様のサーバで最もよく用いられているRed Hat Enterprise Linux (Rhel)を中心に基礎的なコマンドを紹介したいとおもいます。
Linuxコマンドもたくさんあり、オプションを足して行くことできめ細かく設定を変更できるので今回は現場で使えそうなものに限定します。


##◯ここら辺が基本的に覚えておくと使えます


```
$ cd ;ディレクトリ移動
$ cd ~ ;ルートへ移動
$ cd .. ;１つ上の階層に移動する
$ cd ../.. ; 2つ上の階層に移動する
```



```
$ mv ［ファイル］/home ;home配下にファイルを移動する
$ cp ［ファイル］ /home ;home配下にファイルをコピーする
```



```
$ mkdir ［ディレクトリ名］ ;ディレクトリを作る
$ rmdir ［ディレクトリ名］ ;空ディレクトリを消す
$ rm -rf ［ディレクトリ名］;ファイルがあるディレクトリまるごと消す（rm は通常の削除コマンド+-rリカーシヴオプションを足して再回帰的に削除したもの）
```



```
$ yum install [パッケージ名];レポジトリに登録されているパッケージのインストール
$ yum localinstall [パッケージファイル.rpm];自分で落として来たパッケージを直接インストールする
$ yum reinstall [パッケージファイル.rpm];パッケージの再インストール
$ yum erase [パッケージファイル.rpm];パッケージを削除する
$ yum update ;パッケージファイルを更新する
```



```
$ sudo [コマンド] ;スーパーユーザで実行する
$ su [ユーザ変更する] ;
$ su - ; rootユーザに変更する
```



```
$ shutdown -h または $ halt ;サーバをシャットダウンする
$ shutdown -r または $ reboot ;サーバを再起動する
$ kill ;プロセスを終了させる
$ ps -rf ;実行中のプロセスを表示させる
```



```
$ ls ; ディレクトリ内のファイル一覧を見る（-ltr などのオプションで表示変更）
$ cat ; ファイルの中身を確認する
$ pwd ;現在のディレクトリの位置を確認する
```


```
$ vi ［ファイル名］; viエディッタでファイルを編集する。（ファイルがない場合は新規作成となる）

『
i ;編集モードに移行する
r ;置き換える
:w ;保存する
:q ;保存しないで終了する
:wq ;保存して終了する
:q! ;変更がある場合に保存しないで終了する
:wq! ;変更がある場合に保存して終了する
:23 ;23行目に移動する
/ ;検索語 ;前方検索
? ;検索語 ;後方検索
1G ;文頭へ
G ;文末へ
dd ; 1行カット
yy ;1行コピー
p ; ペースト
Escキー ;編集モードを終了する
』
```


```
$ nano ;nanoエディッタで編集する
```



```
$ vi /etc/sysconfig/network-scripts/ifcfg-eth0

『
DEVICE=eth0
#dhcpを指定する場合はdhcp そうでなければ none
BOOTPROTO=dhcp
#no にするとethはダウンする
ONBOOT=yes
#ネットマスクを〜ビットに指定する
NETMASK=
#ipアドレスを指定する
IPADDR=
USERCTL=no
』
```



```
$ ssh ［ホスト名 またはIP］sshで入る。
$ telnet ［ホスト名 またはIP］telnetで入る（セキュリティの観点から基本的には非推奨）
```



```
$ grep ［正規表現］; 〜が含まれる行を抜き出す（通常はcatやps などのコマンドの後に オプション＋ファイル名 | grep error でエラーが含まれる箇所をなどのようにつける）
$ awk ‘{print $1}’ ; 最初の1列目を抜き出す （通常はcatやps などのコマンドの後に オプション＋ファイル名 | awk ‘{print $1}’ ;などのようにつける）
```



```
$ wget ［URL先のDLするファイル］ ; リンク先のファイルをダウンロードする
$ scp [ログイン名@ホスト名またはIP]:/home [ログイン名@ホスト名またはIP]:/home ;リモートマシン間のファイルを移動する（両機の/homeから/home）という設定
$ curl ;ファイルをアップロードしたりpostしたりする
```
